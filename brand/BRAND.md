# ForkCart Brand Guide

## Logo

- **Primary:** `logo.png` (cropped, transparent-ready)
- **Small:** `logo-sm.png` (400px wide)
- **Style:** Git-fork icon — a fork that doubles as a code branch symbol
- **Usage:** Always on white/light backgrounds. Don't stretch, rotate, or recolor.

## Colors

### Primary

| Name         | Hex       | RGB           | Usage                        |
| ------------ | --------- | ------------- | ---------------------------- |
| **Charcoal** | `#31363c` | 49, 54, 60    | Logo, headings, primary text |
| **Slate**    | `#282d34` | 40, 45, 52    | Dark backgrounds, footer     |
| **Mist**     | `#c8c9cb` | 200, 201, 203 | Borders, subtle dividers     |

## Typography

### Fonts

- **Headings:** Inter (700, 600)
- **Body:** Inter (400, 500)
- **Code:** JetBrains Mono or Fira Code

### Scale

- Hero: 48px / 3rem
- H1: 36px / 2.25rem
- H2: 24px / 1.5rem
- H3: 20px / 1.25rem
- Body: 16px / 1rem
- Small: 14px / 0.875rem
- Caption: 12px / 0.75rem

## Components

### Buttons

- **Primary:** bg-[#10b981] text-white, rounded-lg, px-6 py-3
- **Secondary:** border border-[#31363c] text-[#31363c], rounded-lg
- **Ghost:** text-[#10b981] hover:bg-[#d1fae5], rounded-lg

### Cards

- bg-white, rounded-xl, shadow-sm, border border-[#e5e7eb]
- Hover: shadow-md, slight translate-y

### Badges

- bg-[#d1fae5] text-[#065f46], rounded-full, px-3 py-1, text-sm font-medium

## Voice & Tone

- **Technical but approachable** — we're devs talking to devs
- **Confident, not arrogant** — "the only full-stack TS e-commerce" is a fact, not bragging
- **Short sentences** — no enterprise fluff
- **Show, don't tell** — code snippets over bullet points

## Don'ts

- ❌ No gradients on buttons
- ❌ No rounded-full on cards
- ❌ No Comic Sans (obviously)
- ❌ No stock photos of people pointing at screens
- ❌ No "AI-powered" in every headline — it's a feature, not the identity
